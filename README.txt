Thiết lập git commit với tên tiếng việt có dấu và email công ty​

Clone repo về máy cá nhân của bạn​

Sửa nội dung sample1.txt với nội dung tuỳ ý và push lên nhánh master​

(Có thể xảy ra conflict & non-fast-forward)​

Xoá remote hiện tại là origin sau đó them mới remote tên là <origin-nickname> cùng url cũ. Ví dụ: origin-hoang8f​

Tạo và commit file sample2.txt với nội dung sau:​

“Happy saturday​
Create sample 2”​

Push nội dung ở nhánh master ở local lên nhánh master-<nickname> ví dụ master-hoang8f​

Đánh tag ở commit cuối cùng là v1.0-<nickname> và push lên remote repository
